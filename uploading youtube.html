How to Embed a YouTube Video in HTML

Follow these steps to add a YouTube video to your webpage.

Step 1: Find Your YouTube Video

Go to YouTube and open the video you want to embed.

Step 2: Click Share

Below the video, click the Share button.

Step 3: Select Embed

In the share options, choose Embed. This will show you an <iframe> code.

Step 4: Copy the Embed Code

Copy the <iframe> code provided by YouTube.

Step 5: Paste into Your HTML File

Open your HTML file in a text editor and paste the embed code where you want the video to appear.

Step 6: Save and Test

Save your HTML file and open it in a browser. The YouTube video should now be embedded.
